# CRM / Client Management Dashboard UI

##  Overview

  This project is a web-based CRM dashboard UI designed for a small digital marketing agency to 
  manage leads, clients, and daily tasks in one centralized system.

  It helps teams track sales progress, monitor client engagement, and organize follow-ups efficiently.

##  Scenario

  Designed for:
  A 5-member digital agency managing 30+ clients and ongoing leads.


##  Objectives

  * Simplify lead tracking across stages
  * Provide quick business insights
  * Improve follow-up management
  * Organize client and task workflows


##  Key Features

   Dashboard:
  * Summary cards (Leads, Clients, Follow-ups, Tasks)
  * Visual lead pipeline (Kanban style)
  * Client list overview

    Leads Page:
  * Stage-based pipeline tracking
  * Add new lead option

    Client Profile:
  * Client details & project info
  * Activity timeline
  * Notes section

   Task Manager:
  * Task table with status badges
  * Due dates & assignments
  * Filter options


## UX Highlights

  * Color-coded status indicators
  * Clean table layout for scalability
  * Sidebar navigation for quick access



##  Tools Used

  Figma (Design & Prototyping)


##  Impact

  This dashboard helps agencies:

  * Reduce manual tracking
  * Avoid missed follow-ups
  * Manage multiple clients efficiently
